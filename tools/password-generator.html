What makes a password strong?

A strong password has at least 12–16 characters and combines uppercase letters, lowercase letters, numbers, and symbols. The longer and more random, the harder it is to crack.

Never use personal information like names, birthdays, or common words. Each account should have a unique password. Use a password manager to store them safely.
